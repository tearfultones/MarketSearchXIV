# MarketSearchXIV
> A simple CLI tool that allows you to search for items across different worlds or data centers in FFXIV.

## Usage
`MarketSearchXIV.py` takes two arguments: the world/datacenter and the item.

## Example
MarketSearchXIV.py "chaos" "glamour prism"

MarketSearchXIV.py "moogle" "glamour prism"
